Static Index Page

Routes HTTP methods Description
/ GET Render Home Page index.html
Users
/users GET Retrieve multiple or all users
/users/:id GET Retrieve a specific user
/register POST Add users as well as products
/user/login GET Grant a specific user log in access
/users/update PATCH Update a specific user
/user/delete DELETE Delete a specific user
Products
/products GET Retrieve multiple or all products
/products/:id GET Retrieve a specific product
/products POST Add a new product
/products/:id PATCH Update a specific product
/products/delete DELETE Delete a specific product
